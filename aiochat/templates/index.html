Написать чат с браузерным клиентом.

  • поддержка чат-групп с более чем 2мя собеседниками
  • возможность добавляться/удаляться в группу. Можно без администрирования, пользователь сам по имени группы может добавить себя или удалить.
  • Задача со *: реализовать администратора группы.
  • любой пользователь группы должен иметь возможность увидеть что происходило в группе, пока пользователь был в офлайне: сообщения от других пользователей, добавление/удаление пользователей из группы 
  • иметь возможность видеть историю группы для новых пользователей группы

Для реализации WebSocket API использовать асинхронный Python фреймворк (например Twisted, Tornado, Asyncio или другой по желанию)

При реализации подумать о возможности масштабирования системы: какие части системы можно отмасштабировать при росте нагрузки и каким образом.
